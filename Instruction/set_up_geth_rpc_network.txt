Setup up geth rpc network with fixed ether on every node at the beginning

Steps for setup: 
- copy genesis files to all 12 nodes with different account values and fixed 100 ether balance
- init geth on all 12 nodes
- copy the keystore files to all the corrsponding folders (generated by the geth init)
- start geth console with fixed enode value (need to get a list of enode value)
  - see the static_nodes file
- copy the static_node files to all the 12 nodes
- use the following command to make sure that the authentication prompt is bypassed before you run the script. Do this for all other 11 nodes. 
  - ssh-copy-id node6
- use the following command to check tmux console
  - tmux attach -t geth_test_rpc
